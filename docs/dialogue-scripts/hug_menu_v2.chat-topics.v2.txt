# 【成年女仆互动随机文本池｜聊天话题扩展草案 V2】

使用目标：给【沟通 → 聊天】补一层真正像“聊天选项”的话题，不和【讨好 / 夸奖】重合。
使用方向：这些更适合做成【聊天二级菜单】的话题标题，再往下接每个话题自己的随机对白。

适配原则：
- 不写早餐、采买、仓库整理、工具检查等未实现功能。
- 不和【狠狠夸奖】、直球表白型内容重复。
- 重点放在生活感、陪伴感、近况、习惯、心事、关系温度。
- 话题本身要像“聊天选项”，不是“调情按钮”。

建议话题：
- 生活
- 最近
- 心事
- 休息
- 发呆
- 小习惯
- 雨天
- 晚上
- 依赖
- 以后
