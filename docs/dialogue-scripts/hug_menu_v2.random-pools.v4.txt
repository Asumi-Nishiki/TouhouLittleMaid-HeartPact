# 【成年女仆互动随机文本池 V4 完整版】

写法目标：每个好感阶段、每种心情都单独成池。
使用方式建议：按“当前好感阶段 + 当前心情”直接抽取一组完整对白。
占位符：{player}=玩家名字，{maid}=女仆名字。
心情五档：沮丧 / 一般 / 普通 / 开心 / 狗修金LOVE。
沟通结构：主菜单先进入【沟通】，再分流到【聊天 / 讨好 / 讲笑话 / 去给我摇曲柄】。

重要约束：
- 不写早餐、仓库整理、行囊整理、采买、工具补给检查等未实现功能。
- 初始阶段和第一阶段正经、克制，带“请”“您”等客气用语，不用第三人称自称。
- 第二阶段开始逐渐可爱、嘴硬、害羞，偶尔用第三人称自称，加语气词。刚熟悉可以拥抱但不敢主动。
- 第三阶段进入恋人感，更直白但仍会害羞，第三人称自称频率增加。
- 第四阶段新婚，自然撒娇、黏人、软，带语气词，不是命令感。女仆台词和旁白交替出现，不互相抢话。

阶段划分：
- 初始阶段：0~31
- 第一阶段：32~63
- 第二阶段：64~127
- 第三阶段：128~197
- 第四阶段：198+
