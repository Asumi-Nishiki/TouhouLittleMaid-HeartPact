# 小女仆互动 Galgame 风格改写稿 V1

> 小女仆台词关键词：童真、依赖、安全感、撒娇。  
> 注意：她不是恋爱对象，文本要偏亲子互动/照顾感。

---

# 主界面

旁白：  
她抬起头看你，眼睛亮晶晶的，像刚刚发现了一颗属于自己的星星。

小女仆：  
“你来啦！今天也要陪我玩吗？我会乖乖的！”

---

# 摸摸头

旁白：  
你的手掌轻轻落在她发顶。她先是眨了眨眼，很快就像被晒暖的小动物一样眯起眼睛。

小女仆：  
“嘿嘿……这个我喜欢。”

小女仆：  
“再摸一下也可以哦。只要一下下……嗯，两下也可以！”

---

# 举高高

旁白：  
她下意识抓住你的袖口，下一秒又开心地晃起脚尖。世界在她眼里突然变高，风也变得新鲜起来。

小女仆：  
“哇——好高！”

小女仆：  
“我看到好远好远的地方了！你也能看到吗？”

小女仆：  
“再举一会儿好不好？我会抓紧的！”

---

# 让妈妈抱抱

旁白：  
附近的大女仆轻轻靠近，把她稳稳抱进怀里。她的小手揪住衣襟，整个人都慢慢放松下来。

小女仆：  
“暖暖的……”

小女仆：  
“妈妈抱着的时候，就不会害怕了。”

小女仆：  
“我可以再待一小会儿吗？就一小会儿。”

---

# 陪她说话

旁白：  
你蹲下来，把视线降到和她一样的高度。她认真地看着你，像是在努力把每一句话都装进心里。

小女仆：  
“你愿意听我说话吗？”

小女仆：  
“今天我有很努力哦。虽然有一点点害怕，但是没有哭。”

小女仆：  
“所以……可以夸夸我吗？”
