# 【成年女仆互动随机文本池｜聊天话题扩展草案 V1】

使用目标：给【沟通 → 聊天】补一层“可选话题”，避免每次都只是泛聊天。
适配原则：
- 不写早餐、采买、仓库整理、工具检查等未实现功能。
- 话题尽量围绕“陪伴 / 心情 / 关系 / 日常感受 / 轻度调情”展开。
- 初始、熟络阶段偏克制；暧昧阶段开始嘴硬害羞；依恋、相守阶段更自然更黏。
- 这些是“话题入口池”，后续可以再按【阶段 + 心情】继续细拆。

建议可加的话题：
- 问她今天累不累
- 叫她的名字
- 说今天很想见她
- 一起安静待会儿
- 问她要不要靠近一点
- 聊聊小习惯
- 说你今天一直在想她
- 约好晚上再见
- 问她有没有偷偷想你
- 说她今天特别可爱
