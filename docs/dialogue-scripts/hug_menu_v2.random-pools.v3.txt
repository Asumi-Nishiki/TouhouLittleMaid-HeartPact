【成年女仆互动随机文本池 V3】
写法目标：每个好感阶段、每种心情都单独成池。
使用方式建议：按“当前好感阶段 + 当前心情”直接抽取一组完整对白。
占位符：{player}=玩家名字，{maid}=女仆名字。
心情五档：沮丧 / 一般 / 普通 / 开心 / 狗修金LOVE。
沟通结构：主菜单先进入【沟通】，再分流到【聊天 / 讨好 / 去给我摇曲柄】。

重要约束：
- 不写早餐、仓库整理、行囊整理、采买、工具补给检查等未实现功能。
- 初始阶段和第一阶段可以正经、克制。
- 第二阶段开始要逐渐可爱、嘴硬、害羞。
- 第三阶段进入恋人感，可以更直白但仍会害羞。
- 第四阶段是长期关系，可以自然撒娇、偏心、黏人。

阶段划分：
- 初始阶段：0~31
- 第一阶段：32~63
- 第二阶段：64~127
- 第三阶段：128~197
- 第四阶段：198+
