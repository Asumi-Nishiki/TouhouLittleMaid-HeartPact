# 【成年女仆互动随机文本池｜聊天内部随机话题池 V1】

写法目标：
- 不增加聊天二级菜单。
- 点【沟通 → 聊天】后，内部随机命中一个聊天话题，再抽取对应对白。
- 话题标签仅供内部使用，不直接显示给玩家。

使用建议：
- 先按“当前好感阶段 + 当前心情”抽入口聊天文本。
- 若命中【聊天】分支，再从下列聊天话题池中随机命中一个话题。
- 话题命中后，再抽该话题下的一组完整对白。

话题方向约束：
- 不和【讨好 / 狠狠夸奖】重合。
- 不写早餐、仓库整理、采买、工具补给检查等未实现功能。
- 偏生活感、陪伴感、近况、习惯、心事、关系温度。
- 不要把“话题名”直接写进正文里，正文要像自然聊出来的一样。

内部话题标签建议：
- 生活
- 最近
- 心事
- 休息
- 发呆
- 小习惯
- 雨天
- 晚上
- 依赖
- 以后
