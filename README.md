# Touhou Little Maid: Heart Pact

**Touhou Little Maid: Heart Pact**（车万女仆：心契同眠）是一款面向 **Minecraft Forge 1.20.1** 的《车万女仆》恋爱、婚姻与家庭扩展模组。

它把《Touhou Little Maid》从单纯的冒险辅助扩展成一段可以慢慢经营的 GalGame 式情感旅程：从清晨的问候到深夜的闲聊，从随手摸头到认真拥抱，从送出一份心仪的礼物，到终于鼓起勇气说出那句“喜欢”。当感情足够深时，你可以向她表白、交换戒指、结为伴侣，并在婚后的生活中迎来属于你们的小女仆。

## 核心内容

- 日常互动：对话、送礼、摸头、拥抱、亲吻，以及随心情变化的反馈。
- 恋爱阶段：好感阶段、表白、交往、求婚戒指、结婚流程。
- 婚后生活：同床共眠、怀孕、育儿系统。
- 膝枕系统：恋人解锁膝枕，支持第一人称镜头、自由视角、求安慰动作、每日恢复、净化和抗性守护。
- 小女仆系统：成长阶段、取名、妈妈抱娃、小女仆互动、小女仆任务。
- 礼物系统：每日送礼次数、心情和好感反应、礼物分类、拒绝反馈。
- 学习与探险：附魔学、药剂学、战术学、探险等小女仆任务。
- 镜头校准：为不同模型提供本地拥抱/亲吻镜头微调。
- 服务端配置：怀孕天数、成长天数、产后恢复等时间规则由服务端控制。

## 开始游玩

进入游戏后，先打开：

`选项 -> 控制 -> 心契同眠`

检查默认按键：

- 交互按键：`Alt + J`
- 摸头/举高高：`Alt + K`
- 节奏判定键：`J`
- 唤起互动面板：`H`
- 退出膝枕：`Y`

按键绑定以 Minecraft 原版控制设置为准。模组配置界面用于调整称呼、后宫模式、节奏跳过、举高高高度、拥抱距离、调试显示和游戏内指南。

## 好感阶段

本模组沿用《车万女仆》的好感体系，并在恋爱流程中加入关系门槛：

- `0-31`：初始阶段。
- `32+`：摸头相关进展。
- `64+`：拥抱解锁。
- `128+`：表白、交往、亲吻相关内容开始出现。
- `192+`：结婚门槛。
- `384`：原版三级好感点数。

如果某个选项没有显示，通常是因为关系还没有达到对应阶段。

## 成年女仆互动

成年女仆可以进行：

- 沟通
- 摸头
- 拥抱
- 亲吻
- 表白
- 结婚
- 赠送礼物
- 女仆妈妈抱着新生小女仆时的取名/放下相关互动

部分选项会根据心情、归属、婚姻状态、后宫模式、是否抱着孩子、女仆是否成年等条件自动隐藏。

## 快照版更新：膝枕

达到恋人关系后，如果女仆心情在普通以上，可以在互动界面进入膝枕。这个功能更推荐使用第一人称体验：第三视角因为模型、坐姿和摄像机角度差异，比较难一次适配所有女仆模型；第一人称下整体效果会更自然。

不同女仆模型的身高、坐姿、头部骨骼和服装体积都可能不同。如果发现膝枕时视野被挡住，或者画面离女仆太近/太远，请在处于膝枕状态时点击右下角铅笔按钮，调整膝枕摄像机高度和膝枕 FOV 缩放，直到达到你想要的效果。

膝枕界面右上角有隐藏按钮。点击后会隐藏互动 UI，进入自由视角模式；默认按 `H` 可以重新唤起互动面板，默认按 `Y` 可以退出膝枕。`H` 和 `Y` 都可以在 `选项 -> 控制 -> 心契同眠` 中重新绑定。

膝枕也带有每日恢复机制：

- 恋人阶段：每天最多恢复 `10颗心`，并可净化负面效果 `3次`。
- 结婚阶段：每天最多恢复 `15颗心`，并可净化负面效果 `3次`。
- 结婚后额外获得抗性守护：每天最多 `3次`，每次持续 `50秒`。
- 每日恢复、净化和抗性次数由服务端结算，客户端界面只负责显示当前使用状态。

膝枕中还可以选择“求安慰”，女仆会摸摸你的头。这个动作会尽量适配普通模型、GeckoLib 模型和 YSM 模型。

## 亲吻角度校准

不同女仆模型的身高、头部骨骼和镜头位置可能不一样。如果拥抱或亲吻时画面对不到脸，可以使用互动界面的镜头校准功能。

![亲吻角度校准](docs/images/校准.png)

使用方法：

1. 对成年女仆打开互动界面，进入拥抱/亲吻相关界面。
2. 点击右下角退出按钮附近的铅笔按钮。
3. 调整“缩放”控制镜头远近。
4. 调整“上下”控制镜头俯仰。
5. 调好后点击保存。

设置会保存在本地：

`config/maidmarriage/hug-camera.json`

这个配置只影响你自己的客户端，不影响服务器和其他玩家。

## 心情与送礼

女仆心情会影响好感收益和互动反馈。

- 心情低落：正向互动可能不再增加好感。
- 普通心情：正常收益。
- 开心心情：收益提高。
- LOVE 心情：收益最高。

心情不好时继续讲不合适的玩笑、选择冒犯行为、过度互动或送冒犯礼物，会直接降低好感。礼物需要通过互动界面的礼物 UI 赠送，每名玩家每天最多成功送出两次。

礼物大致分为花束、甜食、正餐、贵重物、奇怪礼物和冒犯礼物。求婚戒指、YES 枕头、金蒲公英发卡、族谱工具、调试工具等特殊物品不能作为普通礼物赠送。

## 结婚与怀孕

结婚条件：

- 目标是你的成年女仆。
- 好感达到 `192`。
- 玩家持有两枚未绑定求婚戒指。
- 当前婚姻状态和后宫模式规则没有拦截。

结婚成功后，女仆会收下 YES 枕头。之后女仆睡在女仆床上，玩家在附近原版床满足条件时，可以触发同眠内容。

怀孕规则：

- 普通同眠第一次概率为 `20%`。
- 普通同眠每次失败后，下次概率增加 `5%`，最高 `50%`。
- 连续失败 8 次后，下一次普通同眠必定怀孕。
- 节奏玩法单独按成绩结算，概率从 `0%` 到 `60%`。
- 怀孕有小概率出现双胞胎。

怀孕天数、小女仆成长天数、产后恢复期由服务端 common 配置控制。

## 小女仆

孩子出生后会成为小女仆。出生第一天，小女仆还太小，不能放到地上，需要由妈妈抱着。

小女仆互动包括：

- 摸摸头
- 举高高
- 让妈妈抱抱
- 陪她说话
- 送她礼物
- 满足条件时进行一次性取名

举高高状态下，小女仆会免疫方块窒息伤害。玩家鞘翅滑翔时，也可以继续带着小女仆飞。

## 小女仆任务

小女仆任务只会在女仆 `WORK` 日程中推进。坐下、睡觉和休息时间会暂停任务。

- 附魔学：主手或背包需要有书本。
- 药剂学：主手或背包需要有玻璃瓶或药水。
- 战术学：主手或背包需要有武器。
- 探险：主手或背包需要有木棍。

心情会影响学习成果。小女仆太累时继续学习，奖励品质会下降，并可能降低好感。探险在心情过低时会停止。
