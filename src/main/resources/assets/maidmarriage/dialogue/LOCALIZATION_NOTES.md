# 心契同眠台本本地化基准

这份文件用于约束 `dialogue/en_us` 与 `dialogue/ja_jp` 的台本风格，避免后续翻译时把恋爱游戏台词翻成说明文。

## 角色与标签

- 女仆：`Maid:` / `メイド：`
- 旁白：`Narration:` / `地の文：`
- 玩家：`Player:` / `プレイヤー：`
- 小女仆：`little maid` / `小さなメイド`
- 妈妈：`Mom` / `ママ`

## 变量保护

下面这些内容必须原样保留：

- `${maid}`
- `${player}`
- `${player_maid}`
- `${player_child}`
- `${child_entry_speaker}`
- `${child_entry_text}`
- `%1$s`、`%2$s`
- `{maid}`、`{player}`

## 英文风格

- 使用自然的 GalGame/visual novel 语气，不写成系统说明。
- 旁白优先使用第二人称视角，例如 `You gently...`。
- 女仆台词可以柔软、含蓄、稍带撒娇，但避免过度直白。
- 选项标题保持短，描述保持温柔且可读。

## 日文风格

- 语气以恋爱ゲーム台本为准，避免机械直译。
- 女仆台词根据亲密度使用柔らかい言い回し。
- 小女仆台词更幼一点，但不写成夸张幼儿语。
- 旁白用自然叙述，不滥用敬语。
